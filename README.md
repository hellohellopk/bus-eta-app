# 🚌 hkbus-eta

即時香港巴士到站時間查詢 — KMB、CTB、NLB、GMB

## 功能
- 🔍 搜尋路線號碼 / 巴士站名
- 🗺️ 路線地圖顯示
- ⏱️ 即時到站時間 (ETA)
- ⭐ 收藏常用路線與站點
- 💰 車費顯示

## 網址
https://hellohellopk.github.io/bus-eta-app/

## 資料來源
- [hkbus.app](https://github.com/hkbus/hk-bus-crawling) — 路線資料庫
- [data.etabus.gov.hk](https://data.etabus.gov.hk) — KMB ETA
- [rt.data.gov.hk](https://rt.data.gov.hk) — CTB / NLB ETA
- [data.etagmb.gov.hk](https://data.etagmb.gov.hk) — GMB ETA
